# RM Ayam Kalasan Pondok Marisa

Website sederhana untuk restoran RM Ayam Kalasan Pondok Marisa yang menampilkan menu, informasi kontak, dan testimoni pelanggan.

## Deskripsi

Website ini dibuat menggunakan HTML dan CSS untuk mempresentasikan restoran Ayam Kalasan Pondok Marisa. Website ini memiliki fitur:

- Halaman beranda dengan hero section
- Informasi tentang restoran
- Daftar menu dengan harga
- Testimoni pelanggan
- Informasi kontak dan lokasi

## Cara Menjalankan Website

### Metode 1: Menggunakan Live Server (Direkomendasikan)
1. Install ekstensi Live Server di VS Code
2. Klik kanan pada file `index.html`
3. Pilih "Open with Live Server"

### Metode 2: Langsung dari Browser
1. Buka file `index.html` langsung di browser
2. Klik kanan file tersebut dan pilih "Open with" kemudian pilih browser Anda

## File yang Tersedia

- `index.html` - File HTML utama
- `style.css` - File CSS untuk styling

## Hosting di GitHub Pages

Untuk menghosting website ini di GitHub Pages:

1. Buat repository baru di GitHub
2. Upload semua file (`index.html`, `style.css`, dan lain-lain) ke repository
3. Di halaman repository, klik "Settings"
4. Scroll ke bagian "Pages" di sidebar sebelah kiri
5. Di bagian "Source", pilih "Deploy from a branch"
6. Pilih branch `main` dan folder `/root`
7. Klik "Save"
8. Website akan tersedia di `https://username.github.io/nama-repository`
